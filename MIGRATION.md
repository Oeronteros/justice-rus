## Запуск проекта

1. Установите зависимости:
```bash
npm install
```

2. Создайте `.env.local` файл:
```bash
cp .env.example .env.local
```

3. Заполните переменные окружения

4. Запустите dev сервер:
```bash
npm run dev
```

## Отличия от старой версии

1. **Роутинг**: Next.js App Router вместо ручного роутинга
2. **Типы**: Полная типизация через TypeScript
3. **Компоненты**: React компоненты вместо классов
4. **API**: Next.js API Routes вместо Vercel Functions
5. **Стили**: Tailwind CSS 4 вместо 3
6. **Валидация**: Zod вместо ручной проверки
